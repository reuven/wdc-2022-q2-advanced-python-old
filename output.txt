    "# Exercise: `dictdiff`\n",
    "\n",
    "1. Write a function, `dictdiff`, that takes two dicts as arguments.\n",
    "2. The output will be a dict that describes the differences between the two arguments.\n",
    "    - If the arguments have the same key/value pair, we ignore it in the output.\n",
    "    - If the arguments have the same key, but different values, then the output dict will have that key, plus a list of [arg1_val, arg2_val].\n",
    "    - If one has a key, but not both, then the output dict will contain that key, the value for the arg that exists, and `None` where it doesn't exist.\n",
    "    \n",
    "Examples:\n",
    "\n",
    "```python\n",
    "d1 = {'a':10, 'b':20, 'c':30}\n",
    "d2 = {'a':10, 'b':20, 'c':40}\n",
    "\n",
    "dictdiff(d1, d2)  # {'c':[30, 40]}\n",
    "\n",
    "d1 = {'a':10, 'b':20, 'c':30}\n",
    "d2 = {'a':10, 'b':20, 'd':40}\n",
    "\n",
    "dictdiff(d1, d2)  # {'d': [None, 40], 'c': [30, None]}\n",
    "```"
   ]
